This policy covers the `Veyport` name, logos, and other official project
branding used by Veyport and William Yiu.

customary descriptive use.

## Allowed Uses

Without separate permission, you may:

- Refer to this project by name when truthfully describing the original
  project
- Say that your software is `compatible with Veyport` or `forked from
  Veyport`, so long as the statement is accurate and does not imply
  endorsement
- Use limited references to the project name in commentary, reviews,
  comparisons, tutorials, and other factual discussion
- Use screenshots or short documentation excerpts for commentary, review, news,
  or educational purposes, provided they do not imply official endorsement

## Prohibited Uses

Without prior written permission, you may not:

- Use `Veyport` as the product name for a modified or forked version
- Use the Veyport logos or official branding for a modified or forked version
- Use project branding in a way that suggests sponsorship, endorsement,
  partnership, certification, or official status
- Register or use domains, organization names, package names, or social-media
  handles that are likely to cause confusion about whether a project is
  official

## Forks and Modified Versions

If you distribute a modified version of Veyport, you must remove the official
name and branding from that modified version unless you have prior written
permission.

Truthful statements such as `forked from Veyport` or `based on Veyport` are
allowed, but the modified product itself must be clearly presented as
unofficial.

Trademark and branding permission requests may be sent to
[inquiries@veyport.xyz](mailto:inquiries@veyport.xyz). Responses are provided
on a best-effort basis.
